# M29.2 数字触发任务号兼容说明

## 映射

- `pallet_place_target`、字符串 `"1"`、数字 `1`：获取托盘或垛顶纸箱信息。
- `held_box_pose`、字符串 `"2"`、数字 `2`：获取机器人手持纸箱信息。

## 回显规则

`detection.trigger_task_id` 原样回显请求中的 `task_id`：

- 请求数字 `1`，返回数字 `1`；
- 请求字符串 `"1"`，返回字符串 `"1"`；
- 请求 `pallet_place_target`，返回原任务名。

## 配置

```yaml
task:
  communication:
    trigger_tasks:
      place_target: pallet_place_target
      place_target_aliases: [1, "1"]
      held_box: held_box_pose
      held_box_aliases: [2, "2"]
```

即使现场 `/etc/visionops_v3/carton_palletizing.yaml` 尚未加入 alias 字段，代码默认仍兼容 `1/2`。
