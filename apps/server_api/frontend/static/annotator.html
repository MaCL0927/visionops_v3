2. 检测与裁剪参数
目标类别 ID
只裁剪该检测类别；留空表示所有类别都可作为候选。
目标类别名
和类别 ID 二选一即可；同时填写时优先使用类别名。
检测阈值 conf
低于该置信度的检测框不会参与裁剪；数值越高，候选越少但更可靠。
裁剪扩边 padding
0 表示严格按检测框裁剪；0.05 表示四周各扩展框宽/高的 5%。
多框选择策略
每张图只取一个目标框作为 ROI 候选；如果画面中只有一个目标，默认“置信度+面积”即可。
